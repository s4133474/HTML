Unordered Lists are created using the ul tag and list items are created using the li tag.

  <ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
  </ul>
  • Item 1
  • Item 2
  • Item 3

Ordered Lists are created using the ol tag.

  <ol>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
  </ol>
  1. First Item
  2. Second Item
  3. Third Item

Definition Lists are created using the dl tag, with terms defined using the dt tag and their corresponding descriptions using the dd tag.

  <dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
  <dt>JavaScript</dt>
  <dd>A programming language for the web</dd>
  </dl>
